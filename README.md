<h1>アプリケーション概要</h1>
プロジェクト・タスク・習慣を登録し、カレンダー上で管理ができる

<h1>利用方法</h1>
<h2>プロジェクト作成</h2>
1:トップページ(カレンダー表示ページ)下部の「プロジェクトを登録・確認」を押す
2:プロジェクト一覧ページ上部の「プロジェクト新規登録」を押す
3:新規登録画面でプロジェクトの内容(名称・開始日・終了予定日・終了日・メモ・タスク名)を入力し登録する
4:プロジェクト内のタスクは詳細画面で複数ある場合は並び替えが可能で、完了したらチェック
5:完了したプロジェクトは一覧画面でチェックを行う
<h2>タスク作成</h2>
1:トップページ(カレンダー表示ページ)下部の「タスクを登録・確認」を押す
2:タスク一覧ページ上部の「タスク新規登録」を押す
3:新規登録画面でプロジェクトの内容(名称・開始日・終了予定日・終了日・メモ)を入力し登録する
4:完了したプロジェクトは一覧画面でチェックを行う
