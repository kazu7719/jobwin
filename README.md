Jobwin

<h1>アプリケーション概要</h1>
プロジェクト・タスク・習慣を登録し、カレンダー上で管理ができる

<h1>利用方法</h1>
<h2>プロジェクト作成</h2>
<br>1:トップページ(カレンダー表示ページ)下部の「プロジェクトを登録・確認」を押す</br>
<br>2:プロジェクト一覧ページ上部の「プロジェクト新規登録」を押す</br>
<br>3:新規登録画面でプロジェクトの内容(名称・開始日・終了予定日・終了日・メモ・タスク名)を入力し登録する</br>
<br>4:プロジェクト内のタスクは詳細画面で複数ある場合は並び替えが可能で、完了したらチェック</br>
<br>5:完了したプロジェクトは一覧画面でチェックを行う</br>
<h2>タスク作成</h2>
<br>1:トップページ(カレンダー表示ページ)下部の「タスクを登録・確認」を押す</br>
<br>2:タスク一覧ページ上部の「タスク新規登録」を押す</br>
<br>3:新規登録画面でタスクの内容(名称・開始日・終了予定日・終了日・メモ)を入力し登録する</br>
<br>4:完了したタスクは一覧画面でチェックを行う</br>
<h2>習慣作成</h2>
<br>1:トップページ(カレンダー表示ページ)下部の「習慣を登録・確認」を押す</br>
<br>2:習慣覧ページ上部の「習慣新規登録」を押す</br>
<br>3:新規登録画面で習慣の内容(名称・開始日)を入力し登録する</br>
<br>4:1日ごとに完了した習慣は一覧画面でカレンダー上の日付にチェックを行う</br>
<h2>1日のプロジェクト・タスク・習慣の確認</h2>
<br>1:トップページ(カレンダー表示ページ)にアクセス</br>
<br>2:確認したい日付をクリック</br>
<br>3:クリックすると下部に名称が表示され確認できる</br>
